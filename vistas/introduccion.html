"Mini Plataforma de Ciencia de Datos con Prolog + Web"

Prerrequisitos y configuración del entorno de desarrollo

Para el éxito de este proyecto, es fundamental asegurar que el entorno de desarrollo cuente con todas las herramientas necesarias. El desarrollo de sistemas híbridos requiere una configuración que soporte tanto el motor lógico como el servidor de archivos y la lógica del cliente.

¿Qué aprenderás/aplicarás?

✅ Subida de archivo CSV a través de interfaz gráfica
✅ Backend Prolog (API REST)
✅ Dashboard web responsivo
✅ Gráfica dinámica
✅ Estadísticas automáticas en base al modelo predictivo
✅ Interacción con usuario a través de un formulario

Estructura del Proyecto

📁 proyecto_avanzado/        Directorio raíz del sistema.
├── 📄 server.pl             Backend central, gestiona rutas y lógica de predicción.
├── 📄 crear_dataset.pl      Script de generación de datos sintéticos.
├── 📄 dataset.csv           Almacén de datos en formato de texto plano (generado).
└── 📁 web/                  Directorio para recursos del front-end.
    ├── 📄 index.html        Interfaz de usuario principal.
    ├── 📄 script.js         Lógica de comunicación asíncrona (Fetch API).
    └── 📄 style.css         Definiciones estéticas y diseño responsivo.

Instalación de software base

El componente central es SWI-Prolog, una de las implementaciones más completas y mantenidas del lenguaje. Se recomienda la versión 9.3.x o superior. Para la edición de código, Visual Studio Code (VS Code) es la herramienta predilecta.

Es necesario instalar las siguientes extensiones para las herramientas que se utilizarán en este proyecto:

Prolog y VSC
  • • VSC-Prolog
  • • New-VSC-Prolog

En la configuración del usuario (settings.json), se debe especificar la ruta del binario de swipl para habilitar funciones de autocompletado y validación de reglas en tiempo real.

JavaScript
  • • Prettier
  • • ES6 snippets

Javascript será el puente entre las acciones del usuario y los servicios lógicos de Prolog.

HTML CSS
  • • Auto Rename Tag
  • • HTML CSS Support
  • • HTML/CSS/JavaScript Snippets
  • • CSS Navigation

Diseñaremos una dashboard profesional con dos columnas: una para la carga y visualización de datos, y otra para el formulario de consulta.
