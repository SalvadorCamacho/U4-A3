Interfaz Web (Frontend)

En esta sección construiremos la interfaz gráfica del sistema. Aquí el usuario podrá cargar datos, visualizar resultados y consultar predicciones.

  • 📁 Carga de archivos CSV
  • 📊 Visualización con gráficas
  • 🧠 Interacción con el backend
  • 🩺 Formulario de predicción
⚠️ Este archivo corresponde a: web/index.html
📄 Estructura base HTML + librerías
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Prolog Data Science</title>

    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

    <link rel="stylesheet" href="style.css">
</head>

¿Qué hace este bloque?

Define la base del documento HTML y carga las librerías necesarias para el diseño y visualización de datos.

  • Bootstrap: diseño responsive
  • Chart.js: gráficas dinámicas
  • style.css: estilos personalizados
