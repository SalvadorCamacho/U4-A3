Interfaz Web (Frontend)

En esta sección se construye la interfaz completa del sistema, donde el usuario puede:

  • Cargar archivos CSV
  • Visualizar resultados en gráficas
  • Consultar predicciones
  • Entender la lógica del sistema
Archivo: web/index.html
Estructura base HTML
<!DOCTYPE html>
<html lang="es">

<head>
    <meta charset="UTF-8">
    <title>Modelo predictivo con Prolog y JS</title>

    <!-- Bootstrap -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.2/dist/css/bootstrap.min.css" rel="stylesheet">

    <!-- Chart.js -->
    <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
</head>

¿Qué hace?

Define la base del documento HTML y carga librerías externas.

  • Bootstrap → diseño responsive
  • Chart.js → visualización de datos
