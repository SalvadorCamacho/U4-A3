Estilos del sistema (CSS)

En esta sección definimos los estilos visuales de la aplicación (varios estilos ya están trabajados en Boostrap -notar-). Se utiliza un enfoque minimalista para mantener claridad y buena experiencia de usuario.

  • 🧼 Diseño limpio y moderno
  • 📱 Adaptabilidad responsive
  • 🎯 Enfoque en legibilidad
  • 📊 Optimización de visualización
⚠️ Este archivo corresponde a: web/style.css
🧱 Base global del layout
/* =========================
   BASE MINIMALISTA
========================= */
html, body {
    height: 100%;
    overflow-x: hidden;
    background: #eef2f7;
}

¿Qué hace este bloque?

Define el comportamiento base de la página. Se asegura que el layout ocupe toda la pantalla y elimina el scroll horizontal innecesario.

  • height: 100%: ocupa toda la altura
  • overflow-x: hidden: evita scroll lateral
  • background: color suave moderno

🧩 Estilo de tarjetas (Cards)
/* =========================
   CARDS (solo ajuste visual)
========================= */
.card {
    border-radius: 12px;
}

¿Qué hace este bloque?

Mejora el aspecto visual de las tarjetas usando bordes redondeados, alineándose con tendencias modernas de diseño UI.

  • Diseño más amigable
  • Consistencia visual
  • Mejor UX
🔗 border-radius
📊 Canvas responsivo
/* =========================
   CANVAS RESPONSIVO
========================= */
canvas {
    width: 100% !important;
    height: auto !important;
}

¿Qué hace este bloque?

Asegura que las gráficas se adapten correctamente al tamaño del contenedor, especialmente en dispositivos móviles.

  • Gráficas fluidas
  • Evita desbordamientos
  • Mejor visualización
