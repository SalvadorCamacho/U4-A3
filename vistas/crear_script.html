📁 Subida de CSV al servidor
async function subirCSV() {
    const file = document.getElementById("fileInput").files[0];
    if (!file) return alert("Selecciona un archivo primero");

    const formData = new FormData();
    formData.append("data", file);

    const res = await fetch("/upload", { method: "POST", body: formData });
    const data = await res.json();

    if(data.status === "ok") alert("¡Prolog ha cargado el dataset!");
}

¿Qué hace este bloque?

Envía el archivo CSV al servidor Prolog utilizando una petición HTTP POST.

  • Uso de FormData
  • Petición con fetch
  • Respuesta en JSON

🧠 Solicitud de predicción
async function predecir() {
    const inputs = {
        edad: Number(document.getElementById("edad").value),
        fuma: Number(document.getElementById("fuma").value),
        alcohol: Number(document.getElementById("alcohol").value),
        ejercicio: Number(document.getElementById("ejercicio").value)
    };

    const res = await fetch("/predict", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify(inputs)
    });

    const data = await res.json();
    
    document.getElementById("resultado").innerText =
        `Nivel de Riesgo: ${data.riesgo.toUpperCase()}`;

    actualizarGrafica(data.puntaje);
}

¿Qué hace este bloque?

Recoge los datos del formulario y los envía al backend para obtener una predicción de riesgo.

  • Lectura de inputs HTML
  • Conversión a JSON
  • Actualización del DOM

📊 Renderizado de gráfica
function actualizarGrafica(puntaje) {
    const ctx = document.getElementById("grafica").getContext("2d");
    if (chart) chart.destroy();

    const pBajo = puntaje < 40 ? 100 - puntaje : 10;
    const pAlto = puntaje > 60 ? puntaje : 5;
    const pMedio = 100 - (pBajo + pAlto);

    chart = new Chart(ctx, {
        type: 'bar',
        data: {
            labels: ["Alto", "Medio", "Bajo"],
            datasets: [{
                label: '% Probabilidad',
                data: [pAlto, pMedio, pBajo],
                backgroundColor: ["#ff5f5f", "#ffbd5f", "#5fff8b"]
            }]
        }
    });
}

¿Qué hace este bloque?

Genera una gráfica dinámica basada en el puntaje calculado, representando probabilidades visuales.

  • Uso de Chart.js
  • Renderizado dinámico
  • Visualización de datos
